# 图标准备说明

请在此文件夹中放置以下尺寸的图标：

- `icon-16.png` - 16x16 像素
- `icon-48.png` - 48x48 像素  
- `icon-128.png` - 128x128 像素

## 设计建议

- 使用 PNG 格式，透明背景
- 简洁明了，一眼能看出是下载工具
- 建议配色：蓝色/绿色系（代表下载）
- 可以使用即梦/Dreamina 相关的视觉元素

## 在线设计工具

如果你需要设计图标，可以使用：

- [Canva](https://www.canva.com/) - 在线设计工具
- [Figma](https://www.figma.com/) - 专业设计工具
- [IconKitchen](https://icon.kitchen/) - 图标生成器

## 图标准备好后

1. 将三个尺寸的图标放在此文件夹
2. 更新 manifest.json 中的图标路径
3. 重新打包扩展
