# 更新日志

## [1.0.0] - 2026-04-08

### 首次发布

#### 功能特性
- ✨ 支持即梦（Jimeng）和 Dreamina 平台的图片和视频下载
- 🚫 优先获取无水印的高清资源
- 🎯 精准识别详情页的主要媒体内容
- ⚡ 多层解析策略，确保资源获取成功率
- 🔄 完美适配单页应用（SPA）的路由切换

#### 支持的页面类型
- 即梦图片详情页
- 即梦视频详情页
- Dreamina 图片详情页
- Dreamina 视频详情页

#### 技术实现
- 基于 Chrome Manifest V3
- 多源数据解析（官方 API、路由数据、HTML 内联、DOM）
- 智能降级策略
- SPA 路由监听和详情页锁定机制
- 自动 Referer 处理，提升下载成功率
